# 洛克王国：世界 叶天帝 自动开关相机脚本

完全免费，`exe` 脚本请在仓库 `Release` 中下载。

## 功能

- 图形界面启动、暂停/继续、终止脚本
- 按设定流程执行 `I -> 等待 -> I`
- 支持随机延迟范围
- 支持单次小循环次数和大循环等待
- 支持可选的 `4:00` 月卡跳过点击
  - 每次小循环开始前都会检查北京时间；仅当时间落在 `03:58:00` 到 `04:00:00` 之间时，才会启动等待并在 `04:00:10` 自动点击一次
- 支持保存设置和恢复默认设置

## 使用说明

- 启动脚本后，保持游戏窗口为前台
- 如果游戏使用管理员权限运行，本程序也请使用管理员权限运行
- 默认设置已内置在程序中，首次使用不需要额外附带 JSON 配置文件
- 配置会在保存设置后写入程序目录下的 `auto_camera_toggle_config.json`
- 支持单人6叶/双人12叶。
- 单人：首先背靠墙角，骑乘长颈鹿/奇丽花/迪莫/灵巧；打开精灵背包，切换为6叶队伍；将6叶全部扔出；启动脚本。
- 多人：首先队友放出6叶，主控将同乘放在背包1，然后同乘。按P打开背包切换到6爱分享叶背包2，放出6叶。
- 提示：嘴越大的叶产花效率越高。
- 局限性：相机只影响自己放出来的叶，如果有必要，可以建议队友也开一个脚本。
